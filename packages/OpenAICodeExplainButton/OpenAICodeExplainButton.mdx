# OpenAICodeExplainButton

The `OpenAICodeExplainButton` component is a button with an optional progress indicator. It displays a help icon by default, and when clicked, it triggers a provided event.

## Usage

```jsx
import OpenAICodeExplainButton from "./OpenAICodeExplainButton";

const Example = () => (
  <OpenAICodeExplainButton
    clickEvent={() => console.log("button is clicked")}
    progress={null}
  />
);

export default Example;
```
