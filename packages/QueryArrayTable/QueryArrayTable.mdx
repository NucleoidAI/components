# QueryArrayTable

`QueryArrayTable` isa table component that displays a list of items in a grid format. It accepts a `json` prop which is an array of objects and a `pageSize` prop which determines the number of rows per page.

## Usage

Provide an array of items to the `QueryArrayTable` component to display them in a grid format.
